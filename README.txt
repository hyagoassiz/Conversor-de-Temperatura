O que o programa faz?
Converte as seguintes escalas de temperatura:

1) Kelvin para Celsius utilizando a Fórmula: Celsius + 273,15

2) Kelvin para Fahrenheit utilizando a Fórmula: (Kelvin − 273,15) × 9/5 + 32

3) Transformar Celsius para Kelvin utilizando a Fórmula: Celsius + 273,15

4) Celsius para Fahrenheit utilizando a Fórmula: (Celsius × 9/5) + 32

5) Fahrenheit para Kelvin utilizando a Fórmula: (Fahrenheit − 32) × 5/9 + 273,15

6) Fahrenheit para Celsius utilizando a Fórmula: (Fahrenheit − 32) × 5/9
